# 🎄 Escape From Midnight Claus

![Kotlin](https://img.shields.io/badge/kotlin-%2314354C.svg?style=for-the-badge&logo=kotlin&logoColor=white)
![Status](https://img.shields.io/badge/Status-Complete-brightgreen)

Escape from Midnight Claus is a mobile escape-room game developed using Android Jetpack Compose. The game combines logical puzzles, environmental exploration, and interactive storytelling within a dark Christmas theme.

## 🎮 Controls
- **Tap** – Interact with objects / explore
- **Double Tap** – Select or use an item
- **Inventory Panel** – View and manage collected items


---

## 🛠 Tech Stack

- Kotlin  
- Object-Oriented Programming
- Android Studio

## 🚀 How to Run Locall
1. Clone the repository:
```bash
git clone https://github.com/shine7on/EscapeFromMidnightClaus.git
cd EscapeFromMidnightClaus
```
2. Open it in Android Studio
